# OpenClaw 2026.3.7 版本解读

> 发布时间：2026年3月7日  
> 官方发布：[GitHub Release](https://github.com/openclaw/openclaw/releases/tag/v2026.3.7-beta.1)

## 🎯 核心亮点

### 1. Context Engine 插件接口

新增完整的上下文管理插件系统：

- 支持自定义上下文管理策略
- 完整生命周期钩子：`bootstrap`、`ingest`、`assemble`、`compact`、`afterTurn`
- 零行为变更（未配置时使用默认引擎）
- 支持 `lossless-claw` 等第三方插件

### 2. ACP 持久化频道绑定

Discord 和 Telegram 频道绑定现在可以在重启后保留：

```bash
/acp spawn --thread here
/acp spawn --thread auto
```

- 持久化存储绑定关系
- 路由解析和 CLI 支持
- 重启后自动恢复绑定

### 3. Telegram 主题绑定增强

- 支持 Telegram 主题线程绑定
- 接受 Mac Unicode 破折号选项前缀
- 主题内消息自动路由到 ACP 会话
- 可操作的审批按钮

## 🔧 重要改进

### Web UI 国际化

- 新增西班牙语（`es`）支持
- 语言检测和懒加载
- 语言选择器标签优化

### Web 搜索增强

- Perplexity 切换到 Search API
- 新增语言/地区/时间过滤器
- 结构化搜索结果
- 提供商选择步骤

### 工具改进

**TTS OpenAI 兼容端点**

```json
{
  "messages": {
    "tts": {
      "openai": {
        "baseUrl": "https://your-tts-endpoint.com"
      }
    }
  }
}
```

**Slack DM 打字反馈**

- 新增 `channels.slack.typingReaction` 配置
- Socket Mode DM 显示反应式处理状态

## 🐛 重要修复

### 模型兼容性

- 移除已废弃的 `MiniMax-M2.5-Lightning`
- 修复 OpenAI 兼容端点流式使用统计
- 修复 xAI 原生 web-search 冲突

### 平台修复

**Telegram**
- 修复过期 socket 重启保护
- 避免升级后的重启风暴

**LINE**
- 修复 `requireMention` 群组策略
- 支持前缀配置

**Onboarding**
- 修复无头 Linux 守护进程探测
- SSH/VPS 环境兼容性改进

### 性能优化

- QMD mcporter Windows spawn 加固
- 混合搜索 BM25 排序修复
- Brave 语言代码验证

## 🚀 升级建议

### 必读变更

**Gateway 认证模式**

同时配置 token 和 password 时必须显式设置 mode：

```json
{
  "gateway": {
    "auth": {
      "mode": "token",
      "token": "your-token",
      "password": "your-password"
    }
  }
}
```

### 推荐配置

启用 ACP 协议和持久化绑定：

```json
{
  "acp": {
    "enabled": true,
    "backend": "acpx",
    "dispatch": {
      "enabled": true
    }
  }
}
```
