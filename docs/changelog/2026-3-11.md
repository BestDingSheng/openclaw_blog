# OpenClaw 2026.3.11 版本解读：Hunter & Healer Alpha

> 发布时间：2026年3月11日  
> 版本代号：Hunter & Healer Alpha  
> 官方发布：[GitHub Release](https://github.com/openclaw/openclaw/releases/tag/v2026.3.11)

## 🎯 核心亮点

OpenClaw 2026.3.11 是一个重要的稳定性和功能增强版本，代号 **Hunter & Healer Alpha**，专注于提升可靠性、隐私保护和易用性。

### 1. 免费 1M 上下文模型支持

通过 OpenRouter 集成，现在可以免费使用百万级上下文窗口的模型：

- **GPT 5.4** - 解决了中途停止思考的问题
- **Gemini Embedding 2** - 用于记忆系统的向量嵌入
- 更长的对话历史，更好的上下文理解

### 2. OpenCode Go 语言支持

新增 OpenCode Go 支持，扩展了编程语言生态：

- 原生 Go 代码生成和理解
- 与现有 Python、JavaScript 工具链无缝集成
- 适合系统级编程和高性能应用开发

### 3. 安全加固冲刺

本次更新包含多项安全增强：

- 配置验证失败时采用 fail-closed 策略
- 修复 Gateway 认证漏洞
- 增强 WebSocket 安全检查
- 临时目录权限加固（Linux umask 0002 兼容）

## 🔧 重要改进

### Agent 配置增强

**Context Engine 插件接口**

新增 `ContextEngine` 插件槽，支持自定义上下文管理策略：

```json
{
  "plugins": {
    "entries": {
      "lossless-claw": {
        "enabled": true
      }
    }
  }
}
```

- 完整生命周期钩子：`bootstrap`、`ingest`、`assemble`、`compact`
- 零行为变更（未配置时使用默认引擎）
- 支持 `lossless-claw` 等第三方上下文管理插件

**Compaction 后上下文配置**

新增 `agents.defaults.compaction.postCompactionSections` 配置：

```json
{
  "agents": {
    "defaults": {
      "compaction": {
        "postCompactionSections": ["AGENTS.md", "USER.md"]
      }
    }
  }
}
```

### ACP 协议增强

**持久化频道绑定**

- Discord 频道和 Telegram 主题绑定现在可以在重启后保留
- 新增 CLI 命令管理绑定：
  ```bash
  /acp spawn --thread here
  /acp spawn --thread auto
  ```

**Telegram 主题绑定**

- 支持 Telegram 主题线程绑定
- 主题内消息自动路由到对应 ACP 会话
- 可操作的 Telegram 审批按钮

### 多语言支持

**Web UI 国际化**

- 新增西班牙语（`es`）支持
- 改进语言选择器
- 自动语言检测和懒加载

### 工具增强

**Web 搜索改进**

- Perplexity 切换到 Search API
- 新增语言/地区/时间过滤器
- 结构化搜索结果

**TTS 增强**

- 支持 OpenAI 兼容的 TTS 端点
- 新增 `messages.tts.openai.baseUrl` 配置
- 端点感知的指令验证

## 🚀 升级建议

### 必读变更

**Gateway 认证模式**

如果同时配置了 `gateway.auth.token` 和 `gateway.auth.password`，现在必须显式设置 `gateway.auth.mode`：

```json
{
  "gateway": {
    "auth": {
      "mode": "token",  // 或 "password"
      "token": "your-token",
      "password": "your-password"
    }
  }
}
```

### 推荐配置

**启用 ACP 协议**

```json
{
  "acp": {
    "enabled": true,
    "backend": "acpx",
    "defaultAgent": "codex"
  },
  "plugins": {
    "entries": {
      "acpx": {
        "enabled": true,
        "config": {
          "permissionMode": "approve-all"
        }
      }
    }
  }
}
```

**优化 Compaction**

```json
{
  "agents": {
    "defaults": {
      "compaction": {
        "recentTurnsPreserve": 10,
        "postCompactionSections": ["AGENTS.md", "USER.md"]
      }
    }
  }
}
```
